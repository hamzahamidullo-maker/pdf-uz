# PDF Telegram Bot

Bu bot foydalanuvchidan kelgan fayllarni PDFga aylantiradi:  
- Rasm (.jpg, .png)  
- Text (.txt)  
- Word (.docx)  
- Excel (.xlsx)  

Funksiyalar:
- Majburiy obuna kanal tekshiruvi
- Adminga murojaat tugmasi
- Progress bar bilan PDF yaratish
- Bir nechta faylni bitta PDFga birlashtirish
